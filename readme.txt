Список реализованных пунктов - вся базовая часть:
1. Чтение карты лабиринта из текстового файла реализовано с проверкой корректности символьного задания карты: если число символов в строках файла разное либо есть недопустимый символ тайла, либо на карте не обнаружено символа игрока, то выдаётся сообщение об ошибке и программа завершается.
2. Попиксельная отрисовка карты с помощью тайлов: дыра, стена, пол, ловушка, игрок, переход на следующий уровень и выход из лабиринта
3. Движение игрока по карте с учётом того, что он не может проходить сквозь стены и выходить за пределы окна.
4. При попадании в яму или ловушку (шипы) игрок погибает, выводится сообщение о проигрыше и игра заканчивается.
5. Реализовано два уровня лабиринта; при достижении выхода на 1м уровне выводится сообшение о переходе на следующий уровень и затем карта нового уровня; при достижении выхода на 2м уровне выводится сообщение с поздравлением и игра заканчивается.

Замечение. Обработка мыши не производилась. Учитывалось, что в каждый момент времени может быть нажата лишь одна клавиша движения, которая перемещает персонажа по горизонтали или вертикали, но не диагонали.
